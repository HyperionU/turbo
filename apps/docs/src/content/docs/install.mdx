# Overview

Hyperion University has two different CLI tools to set up your workspace and Nitrox.
- hu-config
    - latest / 2.0.0 (workspace)
- hyperioncli
    - latest / 1.6.0 (workspace, Turbo, Nitrox)
    - unstable / 2.0.0-2 (workspace, Turbo, Nitrox, shadcn/ui)

<Aside>
    As of now, use ***hu-config*** or ***hyperioncli (latest)***. Once ***hyperioncli v2.0.0*** is released, ***hu-config*** will be deprecated.
</Aside>
